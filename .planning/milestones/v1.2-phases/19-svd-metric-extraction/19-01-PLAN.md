For each test, construct a torch.Tensor S of known singular values, then verify the metric function returns the expected value.

**Class: TestStableRank**
- S = [4, 2, 1]: stable_rank = (16+4+1)/16 = 21/16 = 1.3125
- S = [5, 5, 5]: stable_rank = (25+25+25)/25 = 3.0 (all equal -> rank)
- S = [10, 0, 0]: stable_rank = 100/100 = 1.0 (rank-1 matrix)
- Mathematical reasoning: stable_rank = ||M||_F^2 / ||M||_2^2 = sum(s_i^2) / s_1^2

**Class: TestSpectralEntropy**
- S = [1, 1, 1, 1]: p = [0.25]*4, entropy = -4*(0.25*log(0.25)) = log(4) approx 1.3863
- S = [10, 0.001, 0.001, 0.001]: nearly all mass on first SV, entropy near 0
- Mathematical reasoning: spectral_entropy = -sum(p_i * log(p_i)) where p_i = sigma_i / sum(sigma)
- Note: implementation uses EPS=1e-12, so verify near-zero SVs don't cause issues

**Class: TestSpectralGaps**
- S = [10, 7, 3, 2, 1]: gap_1_2 = 3, gap_2_3 = 4, gap_4_5 = 1
- S = [5, 5, 5, 5, 5]: all gaps = 0 (uniform singular values)
- Mathematical reasoning: spectral_gap_i_j = sigma_i - sigma_j (absolute difference)

**Class: TestConditionNumber**
- S = [10, 5, 2]: condition = 10/2 = 5.0
- S = [10, 5, 0]: condition = capped at CONDITION_CAP = 1e6 (due to EPS guard)
- S = [1, 1, 1]: condition = 1.0 (identity-like)
- Mathematical reasoning: condition_number = sigma_1 / sigma_n

**Class: TestRank1ResidualNorm**
- S = [5, 3, 1]: ||M||_F = sqrt(25+9+1) = sqrt(35), residual = sqrt(9+1) = sqrt(10)
  - rank1_residual = sqrt(10)/sqrt(35) = sqrt(2/7) approx 0.5345
- S = [10, 0, 0]: residual = sqrt(0)/sqrt(100) = 0 (perfect rank-1)
  - Note: implementation uses EPS, so value will be tiny but near 0
- Mathematical reasoning: ||M - sigma_1 u_1 v_1^T||_F / ||M||_F = sqrt(sum(s_i^2 for i>0)) / sqrt(sum(s_i^2))
- For this test, U and Vh are not used in the computation (only S matters), but must be provided. Create dummy U, Vh of correct shape.

**Class: TestReadWriteAlignment**
- Create U with U[:,0] = [1,0,0,...], Vh with Vh[0,:] = [1,0,0,...]: alignment = |1| = 1.0
- Create U with U[:,0] = [1,0,0,...], Vh with Vh[0,:] = [0,1,0,...]: alignment = |0| = 0.0
- Create U with U[:,0] = [1/sqrt(2), 1/sqrt(2), 0,...], Vh with Vh[0,:] = [1/sqrt(2), 1/sqrt(2), 0,...]: alignment = 1.0
- Mathematical reasoning: read_write_alignment = |u_1 . v_1| where u_1 is top left SV, v_1 is top right SV

All assertions use `torch.testing.assert_close` with atol=1e-5, rtol=1e-5.
