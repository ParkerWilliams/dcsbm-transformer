Create `tests/audit/test_grassmannian_distance.py` following the established audit pattern.

Per user decision: "Verify against geodesic distance definition: d = sqrt(sum(theta_i^2)) where theta_i = arccos(sigma_i), per Edelman et al. (1998). Test with rotation matrices having analytically computable principal angles. Test edge cases: identical subspaces (d=0), orthogonal subspaces (d=pi/2*sqrt(k)), near-degenerate cases. Verify formula for k=1, k=2, k=3."

**Class: TestGrassmannianIdenticalSubspaces**
- Create two identical orthonormal matrices U1 = U2 = torch.eye(4, 2) (4x2)
- Grassmannian distance should be 0.0 (identical subspaces)
- cos_angles = svdvals(eye(2)) = [1, 1], angles = [0, 0], d = 0
- Assert result is 0.0 within atol=1e-6

**Class: TestGrassmannianOrthogonalSubspaces**
- Create U1 with columns [1,0,0,0] and [0,1,0,0], U2 with columns [0,0,1,0] and [0,0,0,1]
- cos_angles = svdvals(zeros(2,2)) = [0, 0], angles = [pi/2, pi/2]
- d = sqrt(2*(pi/2)^2) = pi/2 * sqrt(2) for k=2
- Assert result matches pi/2*sqrt(2) within atol=1e-5

**Class: TestGrassmannianKnownAngle**
- Construct U1 = [[1,0],[0,1],[0,0],[0,0]] and U2 by rotating U1 by a known angle theta = pi/6 in the (e1, e3) plane:
  - U2 = [[cos(pi/6), 0], [0, 1], [sin(pi/6), 0], [0, 0]]
- For k=2: one principal angle is pi/6, other is 0 (e2 unchanged)
- d = sqrt((pi/6)^2 + 0^2) = pi/6
- Assert result matches pi/6 within atol=1e-5

**Class: TestGrassmannianVaryingK**
Per user decision: "Verify formula for k=1, k=2, k=3."
- Create 6x6 identity-like orthonormal matrices, rotate specific axes
- k=1: single principal angle, d = |theta|
- k=2: two principal angles, d = sqrt(theta_1^2 + theta_2^2)
- k=3: three principal angles, d = sqrt(theta_1^2 + theta_2^2 + theta_3^2)
- Construct rotation matrices with known angles for each k and verify

**Class: TestGrassmannianClipping**
- Verify that clipping to [-1, 1] doesn't hide bugs: create nearly-identical subspaces where numerical noise might push cos_angles slightly above 1.0
- Use U1 and U2 = U1 + tiny_perturbation (1e-7 scale), re-orthonormalize
- Verify distance is small but non-negative (not NaN)

**Class: TestGrassmannianBatched**
- Test with batched input: U_prev shape [2, 4, 3], U_curr shape [2, 4, 3], k=2
- Verify output shape is [2] (one distance per batch element)

All tests use `torch.manual_seed(42)` where randomness is needed.
