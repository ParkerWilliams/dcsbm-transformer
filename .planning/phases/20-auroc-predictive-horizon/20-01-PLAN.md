  <files>tests/audit/test_auroc_computation.py</files>
  <action>
Create `tests/audit/test_auroc_computation.py` with the following test classes following the established audit test pattern (descriptive class names, 1-2 line mathematical reasoning comments per assertion, independent algebra checks):

**class TestAurocVsSklearn:**
- Test with overlapping distributions (e.g., violations=[3,4,5,6,7], controls=[1,2,2.5,3.5,4.5]): compute `auroc_from_groups(violations, controls)`, compare against `sklearn.metrics.roc_auc_score(y_true=[1]*5+[0]*5, y_score=concat(violations, controls))`. Assert abs difference < 1e-10.
- Test with widely separated distributions (violations=[10,11,12], controls=[1,2,3]): AUROC should be 1.0. Verify both our function and sklearn agree.
- Test with identical values (violations=[5,5,5], controls=[5,5,5]): AUROC should be 0.5 (chance level). Verify both agree.
- Test with reversed distributions (all violations < all controls): AUROC should be 0.0. Verify both agree.

**class TestAurocVsMannWhitneyU:**
- Same overlapping distribution test as above but compare against `scipy.stats.mannwhitneyu(violations, controls, alternative='greater')`. Compute `U / (n_v * n_c)` and assert match within 1e-10.
- IMPORTANT: Per Pitfall 4 in RESEARCH.md, `mannwhitneyu` returns different U depending on `alternative` parameter. Use `alternative='greater'` so group1 (violations) having higher ranks yields the correct U. If the match fails, try computing `1 - U/(n_v*n_c)` or using `alternative='two-sided'` and taking the minimum U.
- Three-way comparison: for the same inputs, verify our_auroc == sklearn_auroc == mannwhitneyu_auroc (all within 1e-10).

**class TestAurocAnalyticDistributions:**
- Two Gaussians with known separation: draw N(mu1=2, sigma=1) vs N(mu2=0, sigma=1) with large n=10000 each, seeded rng. Theoretical AUROC = Phi(delta / sqrt(2)) where delta = (mu1-mu2)/sigma = 2. Phi(sqrt(2)) approx 0.9214. Assert our AUROC is within 0.02 of theoretical (sampling noise).
- Perfect separation with gap: violations in [10,20], controls in [0,5]. AUROC must be exactly 1.0.

**class TestAurocEdgeCases:**
- Empty violations array -> NaN return
- Empty controls array -> NaN return
- Single element in each group (violations=[5], controls=[3]) -> should compute valid AUROC (1.0 since 5>3)
- Tied values across groups: violations=[3,3,3], controls=[3,3,3] -> AUROC = 0.5 (midrank handling). Verify scipy.rankdata assigns midranks [3.5, 3.5, 3.5, 3.5, 3.5, 3.5] for combined array.

All tests import from `src.analysis.auroc_horizon import auroc_from_groups`. Use `numpy` for array construction, `sklearn.metrics.roc_auc_score` and `scipy.stats.mannwhitneyu` as oracle references.
