def spectral_curvature(spectra, window_length=7, polyorder=3) -> ndarray:
    """kappa = ||a_perp|| / ||v||^2
    where a_perp = a - (a.v_hat)*v_hat

    v = diff(smoothed, axis=0)  # [n_steps-1, k] velocity
    a = diff(v, axis=0)         # [n_steps-2, k] acceleration

    For each t in range(len(a)):
        orig_idx = t + 1  # Maps derivative index to original space
        curvature[orig_idx] = a_perp_norm / speed^2
    """

def spectral_torsion(spectra, window_length=7, polyorder=3) -> ndarray:
    """tau = ||j_perp|| / (||v|| * ||a_perp||)
    where j_perp = j - (j.v_hat)*v_hat - (j.n_hat)*n_hat

    v = diff(smoothed, axis=0)   # velocity
    a = diff(v, axis=0)          # acceleration
    j = diff(a, axis=0)          # jerk

    For each t in range(len(j)):
        orig_idx = t + 2  # Maps derivative index to original space
        torsion[orig_idx] = j_perp_norm / (speed * a_perp_norm)
    """

def smooth_spectrum(spectra, window_length=7, polyorder=3) -> ndarray:
    """Savitzky-Golay smoothing. If n_steps < window_length, returns copy."""

Key implementation details:
- curvature[0] and curvature[-1] are always NaN (boundary)
- torsion[0], torsion[1], torsion[-1] are always NaN (boundary)
- Zero-velocity steps produce NaN curvature
- Zero a_perp_norm produces NaN torsion (straight line)
- Ordering crossing mask sets additional NaN values

Explicitly verify index mapping: a[t] -> orig_idx=t+1 for curvature, j[t] -> orig_idx=t+2 for torsion."

To bypass Savitzky-Golay smoothing: use `window_length=3, polyorder=1` which is the minimal Savitzky-Golay config and with 3-point window and polyorder 1, it effectively does a 3-point moving average on derivatives but is very close to no smoothing. Alternatively, since `smooth_spectrum` returns a copy when n_steps < window_length, we could set window_length to a value larger than n_steps. But for large N (10000), that's impractical. The best approach: set window_length=3, polyorder=1 to minimize smoothing impact, and verify that the asymptotic convergence rate matches O(h).

**Class: TestCircleCurvature**
- Generate a circle in R^k (k=8, to match typical spectrum dimension):
  - s1(t) = offset + r * cos(2*pi*t/N), s2(t) = offset + r * sin(2*pi*t/N), s3..s8 constant
  - offset = 5.0 (keep values positive like real singular values), r = 1.0
  - Known curvature: kappa = 1/r = 1.0 for all interior points
- N = 1000 points, window_length=3, polyorder=1
- Compute spectral_curvature, take interior points (avoid boundary NaN)
- Compute median curvature of non-NaN interior values
- Assert median curvature is within 5% of 1.0 (discrete approximation with 1000 points should be very close)
- Mathematical reasoning: for a circle of radius r, kappa = 1/r everywhere. Discrete forward differences introduce O(h) error where h = 2*pi/N.

**Class: TestCircleTorsion**
- Same circle as above
- Known torsion: tau = 0 for a planar curve
- Compute spectral_torsion, take interior non-NaN values
- Assert all torsion values are near 0 (within atol=0.05 for N=1000)

**Class: TestHelixCurvatureTorsion**
- Generate a helix in R^k (k=8):
  - s1(t) = offset + r * cos(2*pi*t/N), s2(t) = offset + r * sin(2*pi*t/N), s3(t) = offset + pitch*t/N, s4..s8 constant
  - r = 1.0, pitch = 0.5
  - Known curvature: kappa = r / (r^2 + (pitch/(2*pi))^2)
    - Note: helix parameterized by arc length has kappa = r/(r^2 + c^2) where c = pitch/(2*pi)
    - c = 0.5/(2*pi) approx 0.0796
    - kappa = 1.0 / (1.0 + 0.00633) approx 0.9937
  - Known torsion: tau = c / (r^2 + c^2) = 0.0796 / 1.00633 approx 0.0791
- N = 2000 points, window_length=3, polyorder=1
- Assert median curvature within 10% of analytic kappa
- Assert median torsion within 10% of analytic tau (torsion is noisier than curvature)

**Class: TestCurvatureConvergence**

- Verify error_100 > error_1000 > error_10000 (monotonically decreasing)
- Verify approximate O(h) rate: error ratio between successive N should be roughly 10x (since h decreases 10x)
  - Allow ratio between 3x and 30x to account for boundary effects and discrete approximation

**Class: TestCurvatureIndexMapping**
